### Color Formats

TNS supports outputs in Hex, HSL and RGB formats. 

```bash
 tns g #ff7d00 --color-format hsl
--primary-100=hsl(29 100 90);
--primary-200=hsl(29 100 80);
...

 tns g #ff7d00 --color-format hsl
--primary-100=rgb(255 229 204);
--primary-200=rgb(255 203 153);
...
```
